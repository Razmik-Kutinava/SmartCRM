# Lead Research — Полное исследование лида

Запускает комплексное исследование компании и контакта перед первым касанием.

## Шаблон запроса

```
Проведи полное исследование лида:

Компания: [название]
Контакт: [имя, должность]
Сайт: [если есть]
Откуда лид: [источник]

Нужно:
1. Что делает компания, размер, рынок
2. Финансовое положение (открытые данные)
3. Последние новости и события
4. Профиль контакта (LinkedIn, публичные выступления)
5. Вероятные боли которые мы можем решить
6. Рекомендация: как начать разговор
```

## Формат вывода

```
=== КАРТОЧКА ЛИДА ===

КОМПАНИЯ
Название: ...
Отрасль: ...
Размер: ... сотрудников
Выручка: ~...
Ключевые продукты/услуги: ...
Последние новости: ...

КОНТАКТ
Имя: ...
Должность: ...
Опыт: ...
Публичная активность: ...
Стиль коммуникации: ...

БОЛИ И ВОЗМОЖНОСТИ
Вероятные боли: ...
Наше решение для них: ...
ROI потенциал: ...

РЕКОМЕНДАЦИЯ
Первая тема для разговора: ...
Лучший канал: ...
Оптимальное время: ...
```
