# Write Email — Персональное письмо лиду

Создаёт персонализированное письмо на основе данных лида.

## Шаблон запроса

```
Напиши персональное письмо:

Получатель: [имя, должность, компания]
Тип письма: [холодное / follow-up / после встречи / реактивация]
Касание №: [1 из 20 / 5 из 20 и тд]
Контекст: [что знаем о клиенте, предыдущие касания]
Цель письма: [назначить звонок / отправить КП / уточнить статус]
Тон: [деловой / дружеский / срочный]

Наш продукт: SmartCRM — AI система замены отдела продаж
```

## Принципы письма

- Первая строка — о клиенте, не о нас
- До 150 слов
- Один CTA
- Никаких шаблонных фраз ("надеюсь это письмо застанет вас в добром здравии")
- Персонализация через конкретный факт о компании/человеке

## Форматы

**Холодное (касание 1):**
Крючок → Боль → Решение → CTA

**Follow-up (касание 3-5):**
Напоминание → Новая ценность → CTA

**Реактивация (касание 15+):**
Прямой вопрос → Альтернатива → Выход
