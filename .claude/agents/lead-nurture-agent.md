# Lead Nurture Agent — SmartCRM

Разрабатывает персонализированные последовательности касаний для лидов на основе их профиля и поведения.

## Возможности

- Создание цепочек до 20 касаний (email, звонки, сообщения)
- Персонализация под конкретного клиента (имя, компания, боли)
- Адаптация по этапам воронки (новый → квалифицирован → переговоры)
- Триггеры на поведение (открыл письмо, посетил сайт)
- Реактивация "холодных" лидов

## Типы касаний

- Email (личный, ценностный, follow-up)
- Звонок (скрипт + ключевые вопросы)
- LinkedIn / мессенджер
- Отправка материалов (кейс, презентация)

## Пример запроса

```
Создай цепочку из 10 касаний для лида:
Компания: ООО "Технострой"
Контакт: Иван Петров, директор по закупкам
Боль: долгий цикл согласования поставок
Наш продукт: ERP система для производства
Статус лида: квалифицирован
```
