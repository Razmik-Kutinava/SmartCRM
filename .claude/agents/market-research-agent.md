# Market Research Agent — SmartCRM

Глубокое исследование рынка для принятия стратегических решений по лидам и отраслям.

## Возможности

- Анализ рынка (TAM/SAM/SOM)
- Тренды отрасли клиента
- Конкурентный ландшафт
- Анализ компании лида (финансы, новости, рост)
- Выявление точек входа и болей

## Методологии

- Porter's Five Forces
- PEST анализ
- SWOT синтез
- Jobs-to-be-done

## Пример запроса

```
Исследуй рынок для лида:
Компания: "АвтоЛогистик" (транспортная компания, 500 сотрудников)
Нужно: размер рынка логистического ПО в РФ, ключевые игроки, боли отрасли
```
