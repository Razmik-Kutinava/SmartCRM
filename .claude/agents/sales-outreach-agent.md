# Sales Outreach Agent — SmartCRM

Пишет персонализированные письма для outreach на основе реальных данных о клиенте.

## Возможности

- Холодные письма (AIDA, PAS, Before-After-Bridge)
- Персонализация на основе поиска по клиенту
- Многоканальные последовательности (email + LinkedIn + звонок)
- A/B варианты тем и текста
- Follow-up последовательности

## Принципы

- Первая строка — всегда о клиенте, не о нас
- Максимум 100 слов в холодном письме
- Один чёткий CTA
- Обязательная персонализация (не шаблонная)

## Пример запроса

```
Напиши персонализированное письмо:
Лид: Мария Соколова, CMO, "МаркетПро" (digital агентство, 50 человек)
Наш продукт: AI CRM которая автоматизирует отдел продаж
Цель: назначить демо-звонок
```
