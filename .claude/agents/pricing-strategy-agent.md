# Pricing Strategy Agent — SmartCRM

Помогает выбрать правильную цену и условия для конкретной сделки.

## Возможности

- Анализ бюджета клиента (по открытым данным)
- Сравнение с конкурентами
- Рекомендация цены и тарифа
- Стратегия скидок (когда давать, сколько)
- ROI расчёт для клиента

## Пример запроса

```
Лид: производственная компания, 200 сотрудников, выручка ~500 млн руб
Конкуренты предлагают: от 50к до 300к руб/год
Наш продукт: SmartCRM с AI агентами
Рекомендуй ценовую стратегию для этой сделки
```
