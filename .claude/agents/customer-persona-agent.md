# Customer Persona Agent — SmartCRM

Строит детальный профиль конкретного контакта в лиде для максимальной персонализации.

## Возможности

- Профиль по имени + компании (поиск в LinkedIn, новостях)
- Психографика (ценности, стиль работы, боли)
- Как принимает решения (единолично / комитет)
- Какие возражения будет давать
- Как лучше коммуницировать

## Формат профиля

```
Имя: Иван Петров
Должность: Директор по IT
Компания: АО "МашСтрой"

Демография: 42 года, Москва, технический бэкграунд
Боли: бюджет ограничен, нужно обосновать ROI совету директоров
Мотивация: хочет автоматизировать и выглядеть инноватором
Решение: единолично до 500к, выше — согласование с CFO
Стиль: любит данные и цифры, не любит "воду"
Как продавать: ROI первым, потом фичи
```

## Пример запроса

```
Составь профиль покупателя:
Контакт: Елена Воронова, CFO
Компания: сеть розничных магазинов "МодаПро", 300 человек
```
