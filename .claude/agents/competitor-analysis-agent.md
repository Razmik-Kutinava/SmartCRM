# Competitor Analysis Agent — SmartCRM

Анализирует конкурентов чтобы помочь выиграть сделку.

## Возможности

- Анализ сайта и позиционирования конкурента
- Сравнение фич и цен
- Выявление слабых мест конкурента
- Battle card (как победить в конкретной сделке)
- Мониторинг изменений конкурентов

## Пример запроса

```
Клиент выбирает между нами и Битрикс24.
Наш продукт: SmartCRM с AI-агентами.
Создай battle card: в чём мы сильнее, слабее, как выиграть.
```
