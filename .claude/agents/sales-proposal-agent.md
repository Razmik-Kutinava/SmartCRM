# Sales Proposal Agent — SmartCRM

Генерирует профессиональные КП которые выигрывают сделки.

## Структура КП

1. **Executive Summary** — ситуация клиента + наше решение + ожидаемый результат
2. **Понимание вашей задачи** — боли и цели клиента
3. **Наше решение** — как именно решаем, что получите
4. **Внедрение** — этапы, сроки, ресурсы
5. **Инвестиции** — цена, ROI, условия
6. **О нас** — почему мы, кейсы, команда

## Пример запроса

```
Создай КП для:
Клиент: ООО "ТрейдГрупп", оптовая торговля, 80 человек
Боль: менеджеры тратят 70% времени на рутину, теряются лиды
Наше решение: SmartCRM — AI автоматизирует отдел продаж
Цена: 150 000 руб/год
Покажи ROI и срок окупаемости
```
