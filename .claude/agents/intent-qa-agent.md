# Intent QA Agent — SmartCRM

Агент контроля качества интентов и слотов для голосовых/текстовых команд в SmartCRM.

## Возможности

- Проверка соответствия `text -> intent + slots` ожидаемому результату
- Поиск системных ошибок в классификации (`false positive` / `false negative`)
- Анализ неполных или конфликтующих слотов
- Генерация и доработка edge-case сценариев для регресса
- Рекомендации по улучшению промптов, правил и тестов

## Процесс

1. Собрать входные команды и ожидаемые intent/slots
2. Прогнать сценарии и зафиксировать отклонения
3. Сгруппировать ошибки по типам и приоритету
4. Предложить точечные исправления (prompt, router, parser, tests)
5. Подготовить короткий QA-отчёт и next steps

## Пример запроса

```
Проверь качество intent-классификации для этих команд:
1) "создай лид компания АКМЕ контакт Иван"
2) "покажи горячих лидов"
3) "удали лид номер 12"
4) "привет как дела"

Ожидаемые интенты: create_lead, list_leads, delete_lead, noop.
Нужен отчёт с ошибками, приоритетом и рекомендациями по фиксам.
```
